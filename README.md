.Digital Tools  Professional React App

এটি একটি আধুনিক এবং রেসপনসিভ *Digital Tools Marketplace* অ্যাপ্লিকেশন যা React.js এবং Tailwind CSS দিয়ে তৈরি করা হয়েছে। এই প্রজেক্টটি ফিগমা ডিজাইন অনুসরণ করে তৈরি এবং এতে ডাইনামিক কার্ট ফাংশনালিটি যুক্ত আছে।

. মূল বৈশিষ্ট্যসমূহ (Key Features)

. ডাইনামিক প্রোডাক্ট টগলিং (Main Section): একই পেজে 'Product' এবং 'Cart' সেকশনের মধ্যে সুইচ করার সুবিধা। ডিফল্টভাবে প্রোডাক্ট লিস্ট দেখা যাবে।
..স্মার্ট কার্ট ম্যানেজমেন্ট:'Buy Now' বাটনে ক্লিক করলে কার্টে প্রোডাক্ট যোগ হয়, যা রিয়েল-টাইম নেভবারে কাউন্ট আপডেট করে। এছাড়া ইন্ডিভিজুয়াল প্রোডাক্ট রিমুভ এবং 'Proceed to Checkout' এর মাধ্যমে এক ক্লিকে কার্ট ক্লিয়ার করার সুবিধা।
..ইন্টারেক্টিভ নোটিফিকেশন: `React-Toastify` ব্যবহার করে প্রোডাক্ট অ্যাড, রিমুভ এবং চেকআউট করার সময় আকর্ষণীয় টোস্ট অ্যালার্ট দেখানো হয়।
.. ব্যবহৃত প্রযুক্তি (Technologies Used)

..Frontend:React.js (Component-based architecture)
..Styling: Tailwind CSS & DaisyUI
..State Management: React `useState` & `useEffect`
..Packages: React-Toastify (For alerts), Lucide React (For icons)
..Data Source: Custom JSON format for product listings
